开发方常见问题

我是需求方

是否需要注册?

一、选择需要开发的项目类型

二、填写项目名称、需求描述等信息，越详细的描述可以给开发者传递更多项目信息，有利于吸引更多靠谱开发者报名。您也可以增加需求文档附件。

三、填写完成后即进入平台审核环节。平台审核过程中发现有任何违法信息、敏感信息、不具任何意义信息将会无法通过审核。

需求方在平台做项目步骤是什么？

一、在平台发布需求：需求方发布时，应尽量把需求描述写清楚以便报名者理解。

二、需求发布出来后其他开发者可根据需求方的联系方式直接联系沟通。

三、与开发者充分沟通项目需求后，双方达成一致意见，平台只提供需求方的需求信息展示，不参与项目具体需求沟通与管理，请务必与开发方沟通确认需求以及项目计划。

如何进行支付交易？

平台不参与具体的交易支付过程，请进行线下交易，平台不会介入纠纷处理，责任由甲乙双方承担

我是开发者

是否需要注册？

本平台不提供开发者注册功能。

如何接单？

一、浏览并选择需要开发的项目。

二、查看具体的项目细节或需求文档附件，之后可以根据需求方联系方式直接沟通联系，商讨项目事宜。平台只提供需求方的需求信息展示，不参与项目具体需求沟通与管理， 请务必与开发方沟通确认需求以及项目计划。同时，平台不参与具体的交易支付过程，请进行线下交易，平台不会介入纠纷处理，责任由甲乙双方承担。

如何进行支付交易？

平台不参与具体的交易支付过程，请进行线下交易，平台不会介入纠纷处理，责任由甲乙双方承担。
