实际案例

Classic Case

PT - 牙侦探

罗雅光科技是一家以光学在健康领域的应用为核心的科技创新型企业。 扣钉人帮助其开发公司品牌下的口腔观察设备全套App。 App以牙侦探硬件为热点，直连手机，一键开启即可在线观察、检测、分析牙齿状况； 线上定位国内各地口腔专家。

  • 产品功能：牙齿健康诊所。区分用户版和医生版
  • 支持平台：iOS端、Android端、小程序、管理后台
Marco-X 智能鞋垫

树屋科技致力于足部智能设备的设计创新与技术应用研发的公司。业务涉及：数字媒体设计、智能穿戴、体感游戏和数字艺术

  • 产品功能：在App可对智能设备数据进行分析与监控
  • 支持平台：iOS端、Android端
享取网络科技有限公司

享取网络致力于提供各类公共场所的商品、零食和外卖等物品的临时存、取服务，为骑手和用户提高送和取的周转速度和效率。扣钉人依据产品特性，为享取网络开发了微信小程序端，并能根据不同用户进行功能区分。

  • 产品功能：为骑手、用户提供在线开箱存取服务
  • 支持平台：小程序、App、管理后台
am glass-欢乐谷园区AR眼镜

am glass是一家人工智能公司，它一直专注于AI底层算法与AR技术的创新研发，服务于全国欢乐谷各大园区，在解决了排队时间长问题的同时，将AR硬件与软件完美结合。

  • 产品功能：在线预约眼镜、眼镜与小程序交互、绑定眼镜等等
  • 支持平台：小程序、管理后台
